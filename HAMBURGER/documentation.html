Chartbot Documentation

Introduction

This documentation provides an overview of Chartbot, a conversational AI project developed using the Groq API for both chat and Text-to-Speech (TTS). This new version includes a professional redesign and a powerful command system.

Features

  • Modern, professional, and responsive chat interface.
  • AI-generated responses powered by the Groq API.
  • Client-side command system for enhanced control (see "Commands" section below).
  • Password-protected AI Text-to-Speech (TTS) functionality using the Groq API.
  • Choice of multiple AI TTS providers and voices.
  • Browser-based TTS as an alternative.
  • Speech recognition for voice input.
  • Dark mode support.
  • Customizable theme color via the /theme command.

Setup

To start using the chatbot, open the index.html file in your browser. You will need to enter your Groq API key in the settings menu to enable the chatbot and TTS functionality.

To use the AI TTS, you will need to enter a password (123123) to unlock the feature.

Commands

Chartbot supports the following slash commands for quick actions:

  • /help - Shows the list of available commands.
  • /clear - Clears the entire chat history from the view.
  • /theme <color> - Changes the primary theme color of the interface.
    • Available colors: chartreuse, blue, red, purple.
    • Example: /theme blue
  • /system <prompt> - Sets a new system prompt for the AI. This allows you to change the chatbot's personality or instructions on the fly.
    • Example: /system You are a helpful assistant that only speaks in rhymes.
  • /history - Displays the current conversation history (user and assistant messages).
