Chartbot Documentation

Introduction

This documentation provides an overview of Chartbot, a conversational AI project developed using the Groq and Deepgram APIs for both chat and Text-to-Speech (TTS).

Features

  • Fullscreen, responsive chat interface.
  • AI-generated responses powered by the Groq API, using the llama-3.1-8b-instant model.
  • Password-protected AI Text-to-Speech (TTS) functionality using the Groq and Deepgram APIs.
  • Choice of AI TTS providers: Groq and Deepgram.
  • Groq voices: Eleanor-PlayAI and Fritz-PlayAI.
  • Deepgram voices: aura-athena-en and aura-orion-en.
  • Browser-based TTS as an alternative.
  • Speech recognition for voice input.
  • Dark mode support.
  • Customizable theme color ("Forest" theme with chartreuse accents).
  • Animations for messages and buttons.

Setup

To start using the chatbot, open the index.html file in your browser. You will need to enter your Groq API key in the settings menu to enable the chatbot and TTS functionality. To use Deepgram TTS, you will also need a Deepgram API key.

To use the AI TTS, you will need to enter a password (123123) to unlock the feature.
