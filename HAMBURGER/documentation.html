Chartbot Documentation

Introduction

This documentation provides an overview of the Chatbot project named CHARTBOT, developed using the OpenAI API. The chatbot is designed to deliver conversational responses based on user input.

Features

  • Interactive chat interface
  • Ability to send messages and receive AI-generated responses
  • Option to choose from different AI models (work in progress)

Setup

To start using the chatbot, open the index.html file in your browser. You will need to enter your API key in the settings to enable the chatbot functionality.
