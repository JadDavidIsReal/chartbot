Chartbot Documentation

Introduction

This documentation provides an overview of the Chatbot project named CHARTBOT. The chatbot is designed to deliver conversational responses based on user input and includes advanced Text-to-Speech (TTS) capabilities.

Features

  • Interactive chat interface with a full-screen desktop view.
  • Ability to send messages and receive AI-generated responses from Groq.
  • Option to choose from different AI models.
  • Detailed Text-to-Speech (TTS) settings, with options for both browser and AI-powered TTS.
  • Volume control and voice selection for browser TTS.
  • AI model and voice selection for AI TTS.
  • Password protection for AI TTS settings to prevent accidental usage.

Setup

To start using the chatbot, open the index.html file in your browser. You will need to enter your Groq API key in the settings to enable the chatbot functionality.

The TTS settings can be configured in the side menu. To access the AI TTS settings, you will need to enter the password "123123".

Requirements

  • A modern web browser that supports the Fetch API and the Web Speech API.
  • A Groq API key for AI chat and TTS functionality.
