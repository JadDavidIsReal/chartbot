Chartbot Documentation

Introduction

This documentation provides an overview of Chartbot, a conversational AI project developed using the Groq API. The chatbot is designed to deliver fast and natural-sounding responses based on user input.

Features

  • Interactive and responsive chat interface.
  • AI-generated responses powered by the Groq API, using the llama-3.1-8b-instant model.
  • Text-to-Speech (TTS) functionality with two modes:
    • Browser TTS: Uses the browser's built-in speech synthesis.
    • AI TTS: Uses Google's TTS service for more natural voices.
  • Speech recognition for voice input.
  • Dark mode support.
  • Customizable theme color (currently chartreuse).

Setup

To start using the chatbot, open the index.html file in your browser. You will need to enter your Groq API key in the settings menu to enable the chatbot functionality.
